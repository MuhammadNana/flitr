Detailed Description

The MultiRawVideoFileConsumer class.

This consumer makes use of the flitr::RawVideoFileWriter to write multiple video streams to multiple FLITr Custom recordings.

For more information about the custom FLITr recording format look at the documentation on the flitr::RawVideoFileWriter.
