Detailed Description

template<class T>
class flitr::TMultiOSGConsumer< T >

A consumer that takes images from a producer (possibly with many streams) and creates images and textures for consumption in OSG.

Creates an image consumer that passes data to OSG. Format is determined from the producer.

Parameters

template<class T >
void TMultiOSGConsumer::startDiscardThread ( )

Creates a background thread that discards images if getNext() cannot be called fast enough. E.g. when OSG processing rate is lower than the producer rate.

Note
The discard thread is possibly broken because it can release an image that is reserved for reading for display. In other words this could cause undesired chopping of frames!
